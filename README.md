# jwt-authentication

## How to use
1. Install packages include Django
 ```python
    pip install requirements.txt
 ```


2. Run Commands *change name <project_name> and <application_name>
 ```
    django-admin startproject <project_name>

    cd <project_name>

    python manage.py startapp <application_name>

    python manage.py startapp accounts

```


3. Reference this repository projects to project you created

 

## Caution
Django_project in this repository won't work your environment
you have to change some settings in `Django_project/Django_project/settings.py`
to suit  your environment
